# 旅遊小工具 Travel Web App

單一 HTML 檔案旅遊管理工具，無需後端，所有資料儲存於瀏覽器 LocalStorage。

---

## 快速啟動

```bash
cd D:\new_work\travel
python -m http.server 5500
```

開啟瀏覽器：`http://localhost:5500/index.html`

> ⚠️ 不可直接用 `file://` 開啟，需透過 HTTP Server。

---

## 功能頁籤

| 頁籤 | 功能 |
|---|---|
| **PLAN** | 每日行程時間軸、Google Maps 連結 |
| **GUIDE** | 景點深度介紹、OpenStreetMap 嵌入地圖 |
| **WALLET** | 計算機匯率換算、多幣別記帳、照片上傳 |
| **LISTS** | 隨身/托運清單打勾、個人備忘錄 |
| **INFO** | 天氣連結、緊急電話、出入境注意事項 |
| **DASH** | 今日行程、每日花費長條圖、清單進度、資料備份 |

---

## 資料備份

DASH 頁最下方 → **匯出 JSON** 可下載備份檔案  
換裝置或清除瀏覽器前請先備份，再用 **匯入備份** 還原。

---

## 發佈方式

- **Netlify Drop**：拖曳 `index.html` 至 [netlify.com/drop](https://netlify.com/drop)，30 秒取得公開網址
- **GitHub Pages**：push 至 repo，Settings → Pages 啟用
- **區網分享**：查詢本機 IP，手機連 `http://192.168.x.x:5500`

## 下次建立新旅遊版本的提示詞

```
# Role
你是一位精通 HTML5, Tailwind CSS 和 JavaScript 的資深前端工程師。你需要幫我製作一個「單頁式旅遊 Web App (Single File HTML)」，這個網頁不需要後端，所有的資料儲存都使用瀏覽器的 LocalStorage。

# Goal
製作一個適合手機瀏覽、介面精美、功能實用的旅遊小工具，供我的親友在旅行時使用。

# Design Style (視覺風格)
請依照以下風格進行 UI 設計：
**[填入風格，例：沖繩海灘風格，搭配吉卜力龍貓設計元素]**
* 請搭配適合該風格的 Google Fonts 字體。
* 配色請符合上述風格設定。
* 卡片與按鈕的導角、陰影請依照該風格調整。

# Input Data (行程資訊)
這是我的旅遊資訊，請將其整合進網頁內容中：

1. **航班資訊**：
   去程：[航空公司 + 班號，起飛時間 → 抵達時間，例：虎航 IT728，16:15 → 18:40]
   回程：[航空公司 + 班號，起飛時間 → 抵達時間，例：虎航 IT729，10:30 → 11:20]

2. **住宿資訊**：
   [每晚一筆，格式：D幾 & D幾：飯店名稱，地址，電話（選填）
   例：
   D1 & D4：那霸休伊特渡假飯店，那覇市安里2-5-16，+81-98-943-8325
   D2 & D3：Airbnb 讀谷公寓]

3. **每日行程**：
   [每天一段，格式如下：
   MM/DD（星期）
   - 行程項目 1
   - 行程項目 2
   住：飯店或民宿名稱

   以此類推到最後一天]

4. **必備清單**：
   隨身行李：[逐行列出，例：護照、日幣/台幣、行動電源、網卡、耳機...]
   托運行李：[逐行列出，例：換洗衣物、充電器、盥洗用具、個人藥品...]

5. **租車資訊**（若無請填「無」）：
   租車公司：[填入中日文名稱]
   取車地點：[填入，例：小祿車站接送]
   需要攜帶：[填入，例：1.護照 2.台灣駕照 3.日文譯本]
   注意事項：[填入，例：1.還車時汽油加滿 2.保留加油收據]

6. **當地貨幣設定**：
   貨幣代碼：[例：JPY / KRW / THB]
   預設匯率（1 當地幣 = ? TWD）：[例：0.218]

# Functional Requirements (功能需求)
請務必實作以下功能模組，並使用底部導航列切換分頁：

1. **PLAN（行程表）**：
   * 將每日行程視覺化，使用時間軸或卡片呈現。
   * 景點與餐廳加入「開啟 Google Maps」按鈕連結。
   * 重要時間點（如航班、集合）用醒目顏色標示。

2. **GUIDE（深度導覽）**：
   * 針對重點景點撰寫深度文字介紹（歷史、冷知識）。
   * 排版仿雜誌/旅遊手冊風格，易於閱讀。
   * 嵌入 OpenStreetMap iframe 地圖。

3. **WALLET（記帳與匯率）**：
   * 計算機匯率換算器（可自訂匯率，支援算式如 500+200*3）。
   * 記帳功能欄位：品項、日期（預設今日）、金額、幣別（TWD/當地幣下拉選擇）。
   * 照片上傳：使用 Canvas 壓縮為 Base64 縮圖存入 LocalStorage。
   * 顯示記帳歷史、雙幣金額、刪除按鈕。
   * 每日花費長條圖顯示於 DASH 頁。

4. **LISTS（清單）**：
   * 隨身行李 / 托運行李 CheckList（可打勾，狀態持久化）。
   * 個人備忘錄：自由輸入文字，網址自動轉為可點擊連結。

5. **INFO（資訊）**：
   * 天氣預報連結（如日本氣象廳或 Windy）。
   * 緊急聯絡資訊（警察、消防救護車、台灣辦事處）。
   * 出入境注意事項與旅遊禁忌。
   * 實用連結（Google Maps、交通查詢、匯率查詢）。

6. **DASH（儀表板）**：
   * 今日行程提醒（依系統日期自動對應）。
   * 每日花費 Canvas 長條圖（多幣別統一換算為 TWD）。
   * 行前清單完成進度條。
   * 航班速查卡片。
   * JSON 資料匯出／匯入備份功能。

# Technical Constraints (技術限制)
* **Single File**：所有 CSS、JS 寫在同一個 HTML 檔案內。
* **RWD**：Mobile-First 設計，max-width 480px 置中，手機操作流暢。
* **Library**：Tailwind CSS CDN + FontAwesome 6.5 CDN。
* **No Backend**：純前端，LocalStorage 儲存所有資料。
* **LocalStorage Keys**：expenses / memos / carry_on / checked_bag / jpy_rate（或對應幣別）。

請生成完整的 HTML 程式碼。
```
